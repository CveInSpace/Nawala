## 🇮🇩 Bahasa Indonesia

**NawalaChecker** adalah alat pemindai domain multi-threaded yang dirancang untuk mendeteksi pemblokiran konten oleh berbagai ISP dan filter DNS Nawala secara akurat.

### ✨ Fitur Utama
-   **Verifikasi 3 Lapis**: Memastikan akurasi 100% dengan mengecek Google DoH, Simulasi Nawala, dan Injeksi ISP Lokal.
-   **Sangat Cepat**: Menggunakan `ThreadPoolExecutor` untuk pengecekan massal dalam waktu singkat.
-   **Notifikasi Pembaruan**: Otomatis mendeteksi jika ada versi baru di repositori resmi.
-   **Antarmuka Bersih**: Tampilan UI yang rapi dengan banner ASCII yang elegan.

### 🛠️ Cara Penggunaan
1.  Jalankan `python main.py`.
2.  Masukkan satu domain atau path ke file `.txt` yang berisi daftar domain.
3.  Hasil akan muncul secara real-time dan dapat disimpan ke file.

## 🇺🇸 English

**NawalaChecker** is a multi-threaded domain scanning tool designed to accurately detect content blocking by various ISPs and Nawala DNS filters.

### ✨ Key Features
-   **Triple-Layer Verification**: Ensures 100% accuracy by checking Google DoH, Nawala Simulation, and Local ISP Injection.
-   **High Speed**: Utilizes `ThreadPoolExecutor` for bulk checking in a short amount of time.
-   **Update Notifier**: Automatically detects if a new version is available in the official repository.
-   **Clean Interface**: Neat UI appearance with an elegant ASCII banner.

### 🛠️ How to Use
1.  Run `python main.py`.
2.  Input a single domain or a path to a `.txt` file containing a domain list.
3.  Results will appear in real-time and can be saved to a file.
